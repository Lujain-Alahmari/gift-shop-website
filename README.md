# 🎁 GiftShop Website

A responsive online gift shop website developed using PHP and MySQL. The project allows users to browse products, manage their shopping cart, and place orders through a simple and user-friendly interface.

## 🚀 Features

- User registration and login
- Browse gift products
- Shopping cart
- Order management
- Product categories
- Responsive design
- Database integration with MySQL

## 🛠️ Technologies Used

- PHP
- HTML5
- CSS3
- JavaScript
- MySQL

## 📂 Project Structure

```
GiftShop/
│── admin/
│── css/
│── images/
│── includes/
│── js/
│── index.php
│── login.php
│── register.php
│── cart.php
│── gift_db.sql
```

## ⚙️ Installation

1. Download or clone the repository.
2. Copy the project folder into your XAMPP `htdocs` directory.
3. Import `gift_db.sql` into phpMyAdmin.
4. Start Apache and MySQL using XAMPP.
5. Open your browser and visit:

```
http://localhost/GiftShop
```
